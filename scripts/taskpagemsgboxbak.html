下面是一个优化后的冒泡排序算法示例，包含详细注释和规范化的代码风格：

PYTHONdef bubble_sort(arr): """ 对列表进行冒泡排序。 参数: arr (list): 待排序的列表。 返回: list: 排序后的列表。 """ n = len(arr) # 外层循环控制趟数，最多需要 n-1 趟 for i in range(n - 1): # 添加一个标志位用于优化，如果一趟下来没有交换，说明列表已排序完毕 swapped = False # 内层循环控制每趟比较次数，每趟比较 n-i-1 次 for j in range(n - 1 - i): # 如果前一个元素大于后一个元素，则交换 if arr[j] > arr[j + 1]: arr[j], arr[j + 1] = arr[j + 1], arr[j] swapped = True # 如果没有发生交换，则说明列表已经有序，提前退出循环 if not swapped: break return arr # 示例用法 if __name__ == "__main__": unsorted_list = [64, 34, 25, 12, 22, 11, 90] sorted_list = bubble_sort(unsorted_list) print("排序后的列表:", sorted_list)

代码优化与规范说明：

  1. 注释详细：每个代码块前都有注释，说明其功能和目的。
  2. 代码规范：变量命名清晰（arr, n, i, j, swapped），函数名采用小写和下划线的命名规范（PEP 8 指南）。
  3. 性能优化：使用 swapped 标志位减少不必要的循环，当列表已经排序时提前退出。
  4. 模块化：将排序功能封装在 bubble_sort 函数中，使得代码更易于复用和测试。
  5. 主程序块：使用 if __name__ == "__main__": 结构，使得代码可以直接运行，也可以作为模块导入而不自动执行。
