exec() 函数：
Python 内置函数 exec() 用于动态地执行字符串形式的 Python 代码。它可以接受三个参数：
代码字符串：skill.content，是需要执行的代码，这里是技能的代码内容（例如，一个函数定义）。
全局命名空间：globals()，提供全局变量的命名空间。
局部命名空间：local_vars，指定用于存储执行代码过程中创建的局部变量的字典。

# 定义全局变量
x = 10

# 定义全局和局部命名空间
globals_dict = {"x": x}  # 将全局变量 x 放入全局字典
locals_dict = {}         # 局部字典

在 Python 中，exec() 函数允许你通过 globals 和 locals 参数来指定全局和局部命名空间。这两个参数都应该是字典类型。下面是如何使用这两个参数的详细说明和示例。
exec() 的基本语法
exec(object, globals=None, locals=None)

object: 要执行的代码字符串或代码对象。
globals: 一个字典，指定全局命名空间。
locals: 一个字典，指定局部命名空间。
示例：指定 globals 和 locals
1. 使用共享作用域
# 定义全局变量
x = 10

# 定义全局和局部命名空间
globals_dict = {"x": x}  # 将全局变量 x 放入全局字典
locals_dict = {}         # 局部字典

# 使用 exec()，并指定 globals 和 locals
exec("result = x + 5", globals_dict, locals_dict)

# 在 locals_dict 中可以访问到 result
print(locals_dict['result'])  # 输出: 15

2. 修改全局和局部变量
# 定义全局变量
a = 5
globals_dict = {"a": a}
locals_dict = {}

# 使用 exec() 来修改全局变量 a
exec("a += 10", globals_dict, locals_dict)

# 检查 globals_dict 中的 a
print(globals_dict['a'])  # 输出: 15

3. 使用 locals 捕获局部变量
def my_function():
    b = 3
    globals_dict = {}
    locals_dict = {}

    # 使用 exec()，并指定 globals 和 locals
    exec("c = b + 2", globals_dict, locals_dict)

    # b 在 locals_dict 中
    print(locals_dict['c'])  # 输出: 5  (b + 2 = 3 + 2)

注意事项
如果你在 exec() 中尝试访问 locals_dict 中的变量，它只在 locals_dict 中可用，外部作用域无法直接访问。
使用 globals 和 locals 是一种强大的功能，但也要小心，因为它们可能会导致意外的行为，特别是在变量的命名冲突或作用域混淆时。
尽量避免使用 exec() 和 eval()，因为它们可能导致安全问题和代码可读性差，尤其是当代码执行来自不可信的输入时。
总结
通过指定 globals 和 locals 参数，你可以控制 exec() 执行环境中的变量作用域和访问权限。这可以帮助你实现更灵活的代码执行，但同时也需要谨慎使用。
