写一段冒泡排序法

Sam Altman: 2024-03-08 16:04:04


冒泡排序是一种简单的排序算法，它重复地遍历要排序的数列，一次比较两个元素，如果他们的顺序错误就把他们交换过来。遍历数列的工作是重复地进行直到没有再需要交换，也就是说该数列已经排序完成。

以下是一个使用冒泡排序法对一个整数数组进行排序的示例：
代码类型:Python复制代码

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 标记是否发生了交换
        swapped = False
        # 从0到n-i-1，因为最后i个元素已经是有序的
        for j in range(0, n-i-1):
            # 如果当前元素大于下一个元素，交换它们
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                # 标记发生了交换
                swapped = True
        # 如果没有发生交换，说明数组已经是有序的，可以提前结束循环
        if not swapped:
            break

# 示例
arr = [64, 34, 25, 12, 22, 11, 90]
bubble_sort(arr)
print("Sorted array is:", arr)


运行这段代码将输出：

```
Sorted array is: [11, 12, 22, 25, 34, 64, 90]
```
