   - 后端处理完成后，通过 WebSocket 将结果推回地图页面

4. **地图控制**：
   - Electron 主进程可以通过 IPC 发送命令到地图页面
   - 地图页面通过 [handle_command](file://c:\dev\agi-ev\ai-sns-el\scripts\js\interact_python.js#L117-L175) 函数处理来自主进程的命令

 六、避免循环依赖的机制

 6.1 前端 - 事件总线

 使用事件总线实现模块间通信，避免直接导入：

 // core/eventBus.js
 class EventBus {
     on(event, handler) { }      // 订阅事件
     off(event, handler) { }     // 取消订阅
     emit(event, data) { }       // 触发事件
     once(event, handler) { }    // 一次性订阅
 }

 示例：
 // modules/agent/agentHandlers.js
 import eventBus from '../../core/eventBus.js';
 eventBus.emit('message:sent', { content: '...' });

 // modules/sns/snsHandlers.js
 eventBus.on('message:sent', (data) => {
     console.log('New message:', data);
 });

 6.2 后端 - 依赖注入

 使用 FastAPI 的依赖注入避免模块间直接导入：

 # modules/chat/service.py
 class ChatService:
     def __init__(self, km_service: KMService = None):
         self.km_service = km_service

 # modules/chat/dependencies.py
 from fastapi import Depends
 from modules.km.dependencies import get_km_service

 def get_chat_service(km_service = Depends(get_km_service)):
     return ChatService(km_service=km_service)

  // 1. 建立WebSocket连接
  connectWebSocket(clientId) {
      const wsUrl = `ws://localhost:8788/ws/${clientId}`;
      this.wsConnection = new WebSocket(wsUrl);

      // 2. 注册消息处理器
      this.wsConnection.onmessage = (event) => {
          const message = JSON.parse(event.data);
          this.handleWebSocketMessage(message);
      };
  }

  // 3. 发送消息

  后端 (api_server.py:190-248):
  @app.websocket("/ws/{client_id}")
  async def websocket_endpoint(websocket: WebSocket, client_id: str):
      # 1. 接受连接
      await ws_manager.connect(websocket, client_id)

      # 2. 接收并处理消息
      while True:
          data = await websocket.receive_json()
          msg_type = data.get('type', '')

          # 处理不同类型的消息
          if msg_type == 'ping':
              await ws_manager.send_message({'type': 'pong'}, client_id)
          elif msg_type == 'broadcast':
              await ws_manager.broadcast(message)
          elif msg_type == 'map_chat':
              await ws_manager.broadcast(map_chat_message)

  支持的消息类型:
  - ping/pong: 心跳检测
  - broadcast: 广播消息给所有客户端
  - map_chat: 地图聊天消息广播
  - 自定义消息类型
